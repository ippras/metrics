# Tomiuk–Loeschcke distance

$$-\ln \frac{1}{n} \sqrt{\sum x_{ij} \sum y_{ij}}$$

## Description

The Tomiuk–Loeschcke distance between populations $x$ and $y$ is defined by the formula above.
