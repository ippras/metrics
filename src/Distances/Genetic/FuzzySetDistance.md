# Fuzzy set distance

$$\frac{\sum 1_{x_{ij} \neq y_{ij}}}{\sum_{j=1}^n m_j}$$

## Abbreviations

- $D_{fs}$

## Description

The fuzzy set distance between populations $x$ and $y$ is defined by the formula above.
