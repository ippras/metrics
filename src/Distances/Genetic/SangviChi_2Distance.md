# Sangvi $\chi^2$ distance

$$\frac{2}{n} \sum \frac{(x_{ij} - y_{ij})^2}{x_{ij} + y_{ij}}$$

## Description

The Sangvi $\chi^2$ distance between populations $x$ and $y$ is defined by the formula above.
