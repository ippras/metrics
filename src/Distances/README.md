Мне нужно сравнить два объекта по триацилглицеридному профилю, там тоже мы имеем процент триацилглицерида - который представляет собой частоту с которой мы его наблюдаем, в сумме дают 1
могу ли я использовать Genetic Distances из Deza (2016) для стравнения двух объектов?

Общее генетическое расстояние между двумя таксонами представляет собой расстояние между наборами
данных, связанных с ДНК, выбранных для их представления. Среди трёх наиболее популярных генетических
расстояний, перечисленных ниже, стандартное генетическое расстояние Ней предполагает, что различия
возникают в результате мутаций и генетического дрейфа, в то время как хордовое расстояние Кавалли-Сфорца–Эдвардса
и расстояние Рейнольдса–Уира–Кокерхэма предполагают только генетический дрейф.

Популяция представлена вектором x D .xij/ с Pn jD1 mj компонентами,

где
- $n$ — количество рассматриваемых локусов (генов; например: ген цвета глаз).
- $m_j$ — количество аллелей в j-м локусе (варианты этого гена; например: карий, голубой, зеленый)
- $x_{ij}$ — частота i-го аллеля (обозначение состояния гена) в j-м локусе гена (положение гена на хромосоме),

Поскольку $x_{ij}$ — это частота, мы имеем $x_{ij} ≥ 0$ и Pmj
iD1 xij ≤ 1. Пусть P обозначает сумму по всем i и j.
