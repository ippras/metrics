# Czekanowsky–Dice distance

`Numerical`

$$ 1 - \frac{2|X \cap Y|}{|X| + |Y|} = \frac{|X \Delta Y|}{|X| + |Y|} $$

## Abbreviations

- Bray–Curtis distance

## Synonyms

- nonmetric coefficient

## Description

The Czekanowsky–Dice distance is a near-metric on $\{0, 1\}^n$.
