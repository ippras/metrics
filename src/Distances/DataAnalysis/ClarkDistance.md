# Clark distance

$$ \left( \frac{1}{n} \sum \left( \frac{x_i - y_i}{|x_i| + |y_i|} \right)^2 \right)^{\frac{1}{2}} $$

## Description

The Clark distance is a distance on $\mathbb{R}^n$.
