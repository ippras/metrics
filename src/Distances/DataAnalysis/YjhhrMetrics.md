# YJHHR metrics

$$ d(X, Y) = (|X \setminus Y|^p + |Y \setminus X|^p)^{\frac{1}{p}} $$

## Description

The YJHHR metrics are defined for any $p \ge 1$ and two finite sets $X, Y$. Another variant is $d'(X, Y) = \frac{d(X, Y)}{|X \cup Y|}$ if $|X \cup Y| > 0$, and $0$ otherwise.
